# 04-接口命令设计

## 1. 摘要采集

```bash
python -m src.cli collect abstracts --source all --days 30
python -m src.cli collect abstracts --source pubmed --query "genOway"
python -m src.cli collect abstracts --source pubmed --query "BRGSF"
python -m src.cli collect abstracts --doi "10.xxxx/yyyy"
python -m src.cli collect abstracts --pmid "12345678"
```

## 2. Google Scholar Alerts 邮件导入

Google Scholar Alerts 只作为补充发现入口，不作为正式证据。

```bash
python -m src.cli import scholar-alerts --mbox "data/raw/scholar_alerts.mbox"
python -m src.cli import scholar-alerts --eml-dir "data/raw/scholar_alerts/"
```

导入后默认状态：

```text
source = google_scholar_alert
review_status = needs_review
fulltext_status = not_checked
```

## 3. 自动寻找全文

```bash
python -m src.cli resolve fulltext --paper-id 123
python -m src.cli resolve fulltext --doi "10.xxxx/yyyy"
python -m src.cli resolve fulltext --pmid "12345678"
python -m src.cli resolve fulltext --status abstract_fetched --limit 50
```

## 4. 生成外部链接

```bash
python -m src.cli links build --paper-id 123
python -m src.cli links build --status abstract_fetched --limit 100
python -m src.cli links list --paper-id 123
```

外部链接包括：

- DOI；
- PubMed；
- PMC；
- Unpaywall；
- Semantic Scholar；
- OpenAlex；
- Google Scholar 检索；
- 出版社页面；
- CNKI / 万方 / 维普 / SinoMed / NSTL 检索入口。

## 5. 解析全文

```bash
python -m src.cli parse fulltext --paper-id 123
python -m src.cli parse fulltext --status downloaded
python -m src.cli parse fulltext --source manual_upload
```

## 6. 分析论文

```bash
python -m src.cli analyze paper --paper-id 123
python -m src.cli analyze pending --limit 100
python -m src.cli analyze manual
```

## 7. 手动导入全文

```bash
python -m src.cli import manual-fulltext --file "data/raw/manual_uploads/paper.pdf"
python -m src.cli import manual-fulltext --dir "data/raw/manual_uploads/"
python -m src.cli import manual-fulltext --file "paper.pdf" --doi "10.xxxx/yyyy"
python -m src.cli import manual-fulltext --file "paper.pdf" --paper-id 123
python -m src.cli import manual-fulltext --file "paper.pdf" --title "paper title"
```

## 8. CAJ 文件登记

第一版只登记 CAJ，不直接解析。

```bash
python -m src.cli import manual-fulltext --file "paper.caj"
python -m src.cli list conversion-needed
```

导入后状态：

```text
file_type = caj
fulltext_source = manual_upload
fulltext_status = conversion_needed
parse_status = conversion_needed
review_status = needs_review
```

## 9. 中文论文手动录入

```bash
python -m src.cli import chinese-record --title "论文标题" --source cnki --year 2025
python -m src.cli import chinese-record --file "data/raw/chinese_records/records.csv"
```

## 10. 手动更新状态

```bash
python -m src.cli update paper --paper-id 123 --reference-value high
python -m src.cli update paper --paper-id 123 --reference-value medium
python -m src.cli update paper --paper-id 123 --reference-value low
python -m src.cli update paper --paper-id 123 --reference-value exclude

python -m src.cli update paper --paper-id 123 --evidence-level A
python -m src.cli update paper --paper-id 123 --review-status confirmed
python -m src.cli update paper --paper-id 123 --region-relevance china_domestic
python -m src.cli update paper --paper-id 123 --manual-override true
python -m src.cli update paper --paper-id 123 --note "适合用于人源化小鼠模型案例"
```

## 11. 报告导出

```bash
python -m src.cli report weekly
python -m src.cli report monthly
python -m src.cli report custom --from 2026-01-01 --to 2026-06-30
python -m src.cli report high-value
python -m src.cli report review-queue
python -m src.cli report by-model
python -m src.cli report by-disease
python -m src.cli report china-domestic
python -m src.cli report conversion-needed
```

## 12. 初始化与维护

```bash
python -m src.cli init db
python -m src.cli status summary
python -m src.cli status paper --paper-id 123
python -m src.cli retry failed --type fulltext
python -m src.cli retry failed --type parse
python -m src.cli retry failed --type analysis
```

所有批处理命令建议支持：

```text
--dry-run
--force
--limit
--since
--source
```
